# Palearn
사용자 입력 기반 AI 학습 계획·관리 앱 'Palearn'<br>
👉🏼 https://palearn-final-app-3.vercel.app

## 프로젝트 소개 

학습에서 가장 큰 문제는 계획을 세우는 것이 아니라,계획을 실제 학습으로 이어가지 못한다는 점입니다.<br>
Palearn은 이 문제에 주목하여 복잡한 입력을 최소화한 계획 설정 UI와 학습 진행 상황을 한눈에 확인할 수 있는 시각적 피드백을 중심으로 설계되었습니다.<br>
사용자는 Daily / Weekly / Monthly 단위로 학습 계획을 관리하고, 학습 기록과 통계를 통해 자신의 학습 흐름을 점검할 수 있습니다.<br>
또한 친구 기능을 통해 서로의 학습 현황을 공유하고 응원함으로써 학습 동기와 지속성을 강화합니다.<br>
Palearn은 보기만 하는 계획 앱이 아니라 바로 행동하게 만드는 학습 보조 앱을 목표로 합니다.<br>

## 👤 사용자 기능

### 📌 학습 계획 관리
- **Daily / Weekly / Monthly** 학습 계획 설정
- 단계별·선택형 UI를 통한 간편한 계획 입력

### 📊 학습 기록 및 통계
- 학습 완료 여부 기록
- 달력 기반 학습 기록 확인
- 진행률(%) 및 완료 현황 시각화

### 🏠 홈 화면 대시보드
- 오늘의 학습 진행률 요약
- 빠른 학습 진입 제공

### 🤝 친구 기능
- 친구 코드 기반 친구 추가
- 친구의 학습 진행 현황 확인
- 학습 응원 기능을 통한 동기 부여

### 🔔 알림 관리
- 학습 관련 알림 통합 확인
- 새로운 알림 / 이전 알림 구분

### 🔍 검색 기능
- 학습 계획 및 기록 빠른 검색

### ⚙️ 프로필 및 환경 설정
- 사용자 정보 관리
- **다크 모드 지원**으로 시각적 피로 감소
